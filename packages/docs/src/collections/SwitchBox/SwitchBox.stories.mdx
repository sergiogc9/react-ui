# SwitchBox

A composed component to give some extra information related to a Switch component.

It includes:

- A switch component.
- Some children elements. Use the namespaced built-in components `SwitchBox.Chip`, `SwitchBox.Content` and `SwitchBox.Icon`.

## Usage

Example of use (playground):

```tsx
<SwitchBox>
	<SwitchBox.Icon icon="help" styling="outlined" />
	<SwitchBox.Content display="block">
		Sed ut perspiciatis unde omnis iste natus error sit voluptatem accusantium doloremque laudantium.
		<SwitchBox.Chip ml={1} variant="green">
			lorem
		</SwitchBox.Chip>
		<SwitchBox.Chip ml={1} variant="blue">
			ipsum
		</SwitchBox.Chip>
	</SwitchBox.Content>
</SwitchBox>
```
