# Image

The Image component is a simple wrapper for an <img /> component with an extra skeleton behavior added when image is loading.

Most important features:

- Same props as the `<img />` component.
- Shows an skeleton while loading.

## Sizes

Use `size`, `width` and / or `width` props to size the image:
