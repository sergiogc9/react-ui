# TextArea

This is an auto-grow `TextArea`. It will grow depending on how much content it has.

By default it will allow two rows, and then It will grow automatically. If you remove the content it will get narrow.

<Story
	name="playground"
	args={{ label: 'Enter some text', maxLength: 50 }}
	argTypes={{
		onBlur: { action: 'blurred' },
		onChange: { action: 'changed' }
	}}
	parameters={getPlaygroundStoryParams()}
>
	{args => <TextArea {...args} />}
</Story>

## Character counter

If you pass the `maxLength` prop to a textarea, this will allow you to write that certain numbers of characters.

Also a character counter will be shown under. This character counter, will be hightlighted in `warning` color when you reach the 90% of the max-length, and in `error`color when you reach the 100%.

<Canvas>
	<Story name="character counter" parameters={getFixedStoryParams()}>
		<TextArea maxLength="40" placeholder="Enter some text" title="custom size" />
	</Story>
</Canvas>

## Custom Size

You can get a custom fixed size providing either the `size`, `height` or `width` prop.

<Canvas>
	<Story name="custom size" parameters={getFixedStoryParams()}>
		<TextArea size="250px" placeholder="Enter some text" title="custom size" />
	</Story>
</Canvas>

## Labels and placeholders

You can pass a label using the `label` prop and a placeholder using the `placeholder` prop.

Labels can be positioned inside or outside the input element using the `labelPosition` prop:

#### With label inside:

<Canvas>
	<Story name="with label inside" parameters={getFixedStoryParams()}>
		<TextArea label="Description" labelPosition="inside" />
	</Story>
</Canvas>

#### With label outside:

<Canvas>
	<Story name="with label outside" parameters={getFixedStoryParams()}>
		<TextArea label="Description" labelPosition="outside" />
	</Story>
</Canvas>

#### With placeholder:

<Canvas>
	<Story name="with placeholder" parameters={getFixedStoryParams()}>
		<TextArea label="Description" placeholder="Enter text" />
	</Story>
</Canvas>

## Helper text and counter

#### With helper text:

A helper text under the input can be shown using the `helperText` prop.

<Canvas>
	<Story name="with helper text" parameters={getFixedStoryParams()}>
		<TextArea helperText="You can type unlimited characters" label="Description" />
	</Story>
</Canvas>

#### With characters counter:

Also the number of characters can be limited using the `maxLength` prop:

<Canvas>
	<Story name="with counter" parameters={getFixedStoryParams()}>
		<TextArea helperText="Type up to 20 characters" label="Description" maxLength={20} />
	</Story>
</Canvas>
