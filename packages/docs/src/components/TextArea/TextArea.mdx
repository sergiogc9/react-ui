# TextArea

This is an auto-grow `TextArea`. It will grow depending on how much content it has.

By default it will allow two rows, and then It will grow automatically. If you remove the content it will get narrow.

<Canvas of={TextAreaMeta.Playground} />

## Character counter

If you pass the `maxLength` prop to a textarea, this will allow you to write that certain numbers of characters.

Also a character counter will be shown under. This character counter, will be hightlighted in `warning` color when you reach the 90% of the max-length, and in `error`color when you reach the 100%.

## Labels and placeholders

You can pass a label using the `label` prop and a placeholder using the `placeholder` prop.

Labels can be positioned inside or outside the input element using the `labelPosition` prop:
