import { ArgsTable, Meta, Story, Canvas } from 'storybook/utils/components';
import { getFixedStoryParams, getPlaygroundStoryParams } from 'storybook/parameters';

import Divider from 'components/Divider';
import Flex from 'components/Flex';

<Meta title="Components/Divider" component={Divider} />

# Divider

The divider will help you organize better your content. Really useful when you have a lot of content on your page.

<Story name="playground" parameters={getPlaygroundStoryParams()}>
	{args => (
		<Flex alignItems="center" height="50px" justifyContent="center" width="800px">
			<Divider {...args} />
		</Flex>
	)}
</Story>

## Default behavior

By default, it will adapt automatically to the width of your container. You can use a padding to trim width from X or Y axis.

<Canvas>
	<Story name="default" parameters={getFixedStoryParams()}>
		<Flex width="800px">
			<Divider />
		</Flex>
	</Story>
</Canvas>

## Vertical behavior

You can convert the divider into a vertical divider using the `isVertical` prop. You can use a padding to trim width from X or Y axis.

<Canvas>
	<Story name="vertical" parameters={getFixedStoryParams()}>
		<Flex height={50} width={20}>
			<Divider isVertical />
		</Flex>
	</Story>
</Canvas>

## Custom divider

You can customize it giving the color, height, etc.. that you need. So you can be as creative as you want.

<Canvas>
	<Story name="custom color" parameters={getFixedStoryParams()}>
		<Flex width="800px">
			<Divider backgroundColor="red.500" height="2px" />
		</Flex>
	</Story>
	<Story name="creative color" parameters={getFixedStoryParams()}>
		<Flex width="800px">
			<Divider background="linear-gradient(90deg, rgba(2,0,36,1) 0%, rgba(9,9,121,1) 16%, rgba(0,212,255,1) 50%)" />
		</Flex>
	</Story>
</Canvas>
