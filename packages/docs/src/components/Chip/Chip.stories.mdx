# Chip

Chips are small elements to display tags, links or useful information.

<Story
	name="playground"
	args={{
		variant: 'blue',
		aspectSize: 's',
		children: 'Click me',
		iconLeft: 'location',
		iconRight: 'close',
		showLeftIcon: true,
		showRightIcon: true
	}}
	argTypes={{
		aspectSize: {
			control: {
				type: 'radio',
				options: ['s', 'm']
			}
		},
		variant: {
			control: {
				type: 'radio',
				options: ['white', 'blue', 'grey', 'yellow', 'red', 'green']
			}
		}
	}}
	parameters={getPlaygroundStoryParams()}
>
	{({ children, variant, aspectSize, iconLeft, iconRight, showLeftIcon, showRightIcon }) => (
		<Chip variant={variant} aspectSize={aspectSize}>
			{showLeftIcon && <Chip.Icon styling="filled" icon={iconLeft} location="left" />}
			<Chip.Label>{children}</Chip.Label>
			{showRightIcon && (
				<Chip.Icon styling="outlined" icon={iconRight} location="right" onClick={() => console.log('TEST')} />
			)}
		</Chip>
	)}
</Story>

## Chip Label with Icon

Chip has the possibility to integrate icons, they are all passed via children. Remember to choose their position using the `location` prop.

<Canvas>
	<Story parameters={getFixedStoryParams()} name="chip + icon" decorators={[ChipDecorator]}>
		{['s', 'm'].map(size =>
			['white', 'blue', 'grey', 'yellow', 'red', 'green'].map(variant => (
				<Chip variant={variant} aspectSize={size}>
					<Chip.Icon styling="filled" icon="bookmark" location="left" />
					<Chip.Label>Bookmarked</Chip.Label>
				</Chip>
			))
		)}
	</Story>
</Canvas>
