# Avatar

A component used to render an avatar.

## Content

The Avatar will display (from more to less priority):

- An image if you provide the `src` prop. If the image fails to load, will jump to the next case.
- Name and surname initials.
- An icon

## AspectSize prop

The prop `aspectSize` receives a `string` value.

Based on that value it changes the `width and height` styles of the Avatar component. Also, you can provide a `size` (in pixels) for a custom size.

All sizes icons:

	<Story of={AvatarMeta.s} />

</Canvas>
