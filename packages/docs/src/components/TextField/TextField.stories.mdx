## Content

You can add content to left or right using the `leftContent` and `rightContent` props.

The left content should be used to add informative content and the right content should be used to add actionable or clickable content.

<Canvas>
	<Story name="only input" parameters={getFixedStoryParams()}>
		<TextField label="Enter some text" />
	</Story>
	<Story name="with left icon" parameters={getFixedStoryParams()}>
		<TextField label="Enter some text" leftContent={<Icon styling="outlined" icon="help" />} />
	</Story>
	<Story name="with right icon" parameters={getFixedStoryParams()}>
		<TextField label="Enter some text" rightContent={<Icon styling="outlined" icon="cloud-upload" />} />
	</Story>
	<Story name="with both icons" parameters={getFixedStoryParams()}>
		<TextField
			label="Enter some text"
			leftContent={<Icon styling="outlined" icon="help" />}
			rightContent={<Icon styling="outlined" icon="cloud-upload" />}
		/>
	</Story>
</Canvas>

## Labels and placeholders

You can pass a label using the `label` prop and a placeholder using the `placeholder` prop.

Labels can be positioned inside or outside the input element using the `labelPosition` prop:

## Helper text and counter

#### With helper text:

A helper text under the input can be shown using the `helperText` prop.

<Canvas>
	<Story name="with helper text" parameters={getFixedStoryParams()}>
		<TextField helperText="You can type unlimited characters" label="Enter some text" />
	</Story>
</Canvas>

#### With characters counter:

Also the number of characters can be limited using the `maxLength` prop:

<Canvas>
	<Story name="with counter" parameters={getFixedStoryParams()}>
		<TextField helperText="Type up to 10 characters" label="Enter some text" maxLength={10} />
	</Story>
</Canvas>

## Custom

You can use some provided custom textfields using the `type` prop:

#### Date

A custom TextField for dates using the `DatePicker` component.

<Canvas>
	<Story name="date" parameters={getFixedStoryParams()}>
		<TextField type="date" label="Select a day" />
	</Story>
</Canvas>

#### Numeric

A TextField for numeric values. It has custom arrows and a default minimum value of `0`.

<Canvas>
	<Story name="number" parameters={getFixedStoryParams()}>
		<TextField type="number" defaultValue={25} label="Enter your age" />
	</Story>
</Canvas>

#### Password

A custom TextField for passwords where password can be shown or hidden.

<Canvas>
	<Story name="password" parameters={getFixedStoryParams()}>
		<TextField type="password" label="Enter your password" />
	</Story>
</Canvas>
