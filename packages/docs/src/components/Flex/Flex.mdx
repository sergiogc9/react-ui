# Flex

The `Flex` component is useful to create layouts using the `flexbox CSS` API. More info about it here:

[https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Flexbox](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Flexbox)<br/>
[https://css-tricks.com/snippets/css/a-guide-to-flexbox/](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)

<StorybookAlert status="info" text="This documentation assumes the knowledge of the Flexbox CSS API." />

The `Flex` component is basically a `Box` component with `display: flex` enabled.

## Basic usage

Using it is really straightforward. Just import it and start using everywhere in your app:

```tsx
import { Flex } from '@typeform/kitt';

const Component = () => {
	return (
		<Flex justifyContent="space-between" py={5} width={300}>
			<Box>First component</Box>
			<Box>Second component</Box>
		</Flex>
	);
};
```
