# Switch

A component used to render a Switch, wich would to toggle the state of a single item on or off.

<Story name="playground" args={{ aspectSize: 'm' }} parameters={getPlaygroundStoryParams()}>
	{args => <Switch {...args} />}
</Story>

## AspectSize prop

The prop `aspectSize` can receive the values `s` or `m`.

Based on that value it changes the `width and height` styles of the Switch component.

All sizes icons:

<Canvas>
	<Story name="s" title="s" parameters={getFixedStoryParams()}>
		<Switch aspectSize="s" />
	</Story>
	<Story name="m" title="m" parameters={getFixedStoryParams()}>
		<Switch aspectSize="m" />
	</Story>
</Canvas>

## Active state

By default, every switch has a `on/off` state and a `hover` that will change its colours and behaviour.

<Canvas>
	<Story name="default" title="default" parameters={getFixedStoryParams()}>
		<Switch aspectSize="m" isDefaultChecked={true} isActive={true} />
	</Story>
</Canvas>

## Disabled switch

You can disable it adding the `isDisabled` prop.

<Canvas>
	<Story name="disabled" title="disabled" parameters={getFixedStoryParams()}>
		<Switch aspectSize="m" isDisabled={true} />
	</Story>
</Canvas>
