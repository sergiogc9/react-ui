# Link

This simple component will apply the link style a the text inside it. It extends all the `Content` component props.

It should be used if you need inside a paragraph or a WYSIWYG. To another cases use the `Button` with the `link` variant.

<Story
  name="playground"
  args={{ children: 'Awesome link' }}
  parameters={getPlaygroundStoryParams()}
>
  {(args) => <Link {...args} />}
</Story>

## Default behavior

<Canvas>
  <Story name="default" parameters={getFixedStoryParams()}>
    <Link>I'm a default link</Link>
  </Story>
</Canvas>

## Opposite behavior

You can pass `opposite` to the `behavior` prop and the underline will be hidden by default and showed on ':hover'.

<Canvas>
  <Story name="opposite" parameters={getFixedStoryParams()}>
    <Link behavior="opposite">I'm an opposite link</Link>
  </Story>
</Canvas>

## Link as `<button>`

You can render a `<button>` tag on the DOM if you create the Link with `as="button"`. The style is the same, though.

<Canvas>
  <Story name="link as button" parameters={getFixedStoryParams()}>
    <Link as="button">I'm a default link</Link>
  </Story>
</Canvas>
