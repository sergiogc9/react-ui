# IconButton

The IconButton component can be used to show a button with only an icon. If you want to use some text, use the `Button` component.

<Story
	name="playground"
	args={{
		aspectSize: 'm',
		icon: 'delete',
		iconStyling: 'filled'
	}}
	argTypes={{
		iconStyling: {
			control: {
				type: 'radio',
				options: ['filled', 'outlined']
			}
		}
	}}
	parameters={getPlaygroundStoryParams()}
>
	{args => (
		<IconButton {...args}>
			<Icon aspectSize={args.aspectSize === 'xl' ? 'l' : args.aspectSize} icon={args.icon} styling={args.iconStyling} />
		</IconButton>
	)}
</Story>

## Sizing

The prop `aspectSize` receives a `string` value.

Based on that value it changes some attributes as its size.

<Canvas>
	<Story name="s" parameters={getFixedStoryParams()}>
		<IconButton aspectSize="s" title="s">
			<Icon aspectSize="s" icon="delete" styling="filled" />
		</IconButton>
	</Story>
	<Story name="m" parameters={getFixedStoryParams()}>
		<IconButton aspectSize="m" title="m">
			<Icon aspectSize="m" icon="delete" styling="filled" />
		</IconButton>
	</Story>
	<Story name="l" parameters={getFixedStoryParams()}>
		<IconButton aspectSize="l" title="l">
			<Icon aspectSize="l" icon="delete" styling="filled" />

	<Story name="xl" parameters={getFixedStoryParams()}>
		<IconButton aspectSize="xl" title="xl">
			<Icon aspectSize="l" icon="delete" styling="filled" />
		</IconButton>
	</Story>
</Canvas>
