# API

You can change the color, distance between pulse elements and size using the props used in Box.

To handle different animation starting points between Pulse elements, use the index prop:

<Canvas>
	<Story name="one" parameters={getFixedStoryParams()}>
		<Spinner>
			<Spinner.Pulse index={1} size="10px" color="primary.500" margin="2.5px" />
		</Spinner>
	</Story>
	<Story name="many same time" parameters={getFixedStoryParams()}>
		<Spinner>
