# Button

Button component.

<Story
	name="playground"
	args={{ aspectSize: 'm', children: 'Click me', variant: 'default' }}
	argTypes={{
		onClick: { action: 'clicked' }
	}}
	parameters={getPlaygroundStoryParams()}
>
	{args => <Button {...args} />}
</Story>

<Story
	name="playground with icons"
	args={{
		aspectSize: 'm',
		children: 'Click me',
		variant: 'default',
		iconLeft: 'cloud-upload',
		iconRight: ''
	}}
	argTypes={{
		onClick: { action: 'clicked' }
	}}
	parameters={getPlaygroundStoryParams()}
>
	{args => (
		<Button {...args}>
			{args.iconLeft && <Button.Icon styling="outlined" icon={args.iconLeft} aspectSize={args.aspectSize} />}
			<Button.Text aspectSize={args.aspectSize} variant={args.variant}>
				{args.children}
			</Button.Text>
			{args.iconRight && <Button.Icon styling="outlined" icon={args.iconRight} aspectSize={args.aspectSize} />}
		</Button>
	)}
</Story>

## AspectSize prop

The prop `aspectSize` receives a `string` value.

Based on that value it changes the `height, font-size, line-height and padding` styles of the Button component.

## Icons

Icons can be added at left or right. For a button with only an icon and no text, please use the IconButton:
