# Alert

Most important features:

- 4 different built-in status variants.
- Customizable content.

## Status variants

Use one of the different `status` available variants:

<Canvas>
	<Story of={AlertMeta.Error} />
	<Story of={AlertMeta.Info} />
	<Story of={AlertMeta.Success} />
	<Story of={AlertMeta.Warning} />
</Canvas>

## Size variants

Use the `aspectSize` prop to use different Alert sizes:

<Canvas>
	<Story of={AlertMeta.Small} />
	<Story of={AlertMeta.Medium} />
</Canvas>

## Customizable content

Use the built-in namespaced components `Alert.Icon` and `Alert.Text` to create custom alert layouts.

Default layout:

<Canvas of={AlertMeta.DefaultLayout} />

Customized layout:
