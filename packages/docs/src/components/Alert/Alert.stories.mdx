# Alert

Most important features:

- 4 different built-in status variants.
- Customizable content.

## Customizable content

Use the built-in namespaced components `Alert.Icon` and `Alert.Text` to create custom alert layouts.

Default layout:

<Canvas>
	<Story name="default layout" parameters={getFixedStoryParams()}>
		<Alert>
			<Alert.Icon />
			<Alert.Text>Some awesome text</Alert.Text>
		</Alert>
	</Story>
</Canvas>

Customized layout:

<Canvas>
	<Story name="custom layout" parameters={getFixedStoryParams()}>
		<Alert flexDirection="column">
			<Alert.Icon my={1} />
			<Alert.Text fontWeight="bold" aspectSize="l">
				This is an important message
			</Alert.Text>
			<Alert.Text>Some awesome text</Alert.Text>
		</Alert>
	</Story>
</Canvas>
