# Badge

Badge is a container for `Badge.Status` and `Badge.Counters`.
Both `Badge.Counter` and `Badge.Status` can be positioned using the `location` prop.

The Badge accept any kind of children but it's main use is with `Icon` and `Avatar`.

<Story
  argTypes={{
    variant: {
      control: {
        type: 'radio',
        options: ['green', 'red', 'yellow', 'blue']
      }
    },
    location: {
      control: {
        type: 'radio',
        options: ['top-left', 'top-right', 'bottom-left', 'bottom-right']
      }
    }
  }}
  args={{
    children: 'John Doe',
    aspectSize: 's',
    variant: 'green',
    location: 'top-right'
  }}
  name="playground-avatar"
  parameters={getPlaygroundStoryParams()}
>
  {({ children, variant, location, withIcon, numberOfItems }) => (
    <Badge>
      <Avatar children={children} aspectSize="s" />
      <Badge.Status variant={variant} location={location} />
    </Badge>
  )}
</Story>
